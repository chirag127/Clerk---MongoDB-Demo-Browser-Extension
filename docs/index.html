ClerkAuth MongoDB Data Browser Extension

A powerful browser extension that seamlessly integrates Clerk authentication with MongoDB, allowing you to browse and manage your database data directly from your browser with secure authentication.

View on GitHub Install Extension

Key Features

Secure Authentication

Powered by Clerk for robust, secure user authentication with JWT tokens and session management.

MongoDB Integration

Seamlessly connect to your MongoDB databases and browse collections, documents, and query data.

Browser Extension

Works directly in your browser as a convenient extension without leaving your development environment.

Installation

From Chrome Web Store

  1. 1. Visit the Chrome Web Store
  2. 2. Search for "ClerkFlow MongoDB Browser"
  3. 3. Click "Add to Chrome"
  4. 4. Grant necessary permissions

From GitHub Releases

  1. 1. Download the latest release from GitHub
  2. 2. Unzip the downloaded file
  3. 3. Open Chrome extensions page
  4. 4. Enable Developer Mode and load unpacked
